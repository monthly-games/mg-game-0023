## 3. 상세 BM 설계

### 3.1 확장/건설 BM

#### 무료 구조
- 기본 자원으로 건설 진행
- 시간 경과로 자연 확장
- 연구 트리 무료 진행

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 건설 즉시 완료 | 시간 스킵 | 프리미엄 재화 |
| 건설 큐 추가 | 동시 건설 +1 | $2.99 |
| 자원 패키지 | 건설 자원 대량 | $4.99~$14.99 |
| 지역 확장권 | 새 지역 즉시 해금 | $6.99~$9.99 |

### 3.2 방치 수익 BM

#### 무료 구조
- 오프라인 자원 수집 상한: 12시간
- 식민지 운영 수익 누적
- 일일 접속 보너스

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 수익 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 수익 3배 패스 | 상시 적용 (30일) | $4.99/월 |
| 오프라인 확장 | 24시간까지 누적 | 패스 포함 |
| 자동 관리 | 위기 자동 대응 | $2.99 (영구) |

### 3.3 프리미엄 콘텐츠 BM

#### 무료 구조
- 기본 난이도 (이지~노멀) 무료
- 표준 시나리오 무료
- 기본 건물/유닛 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 하드코어 모드 | 익스트림/커스텀 난이도 | $2.99 |
| 추가 시나리오 | DLC 스토리/맵 | $4.99~$9.99 |
| 프리미엄 건물 | 특수 건물 해금 | $1.99~$3.99 |
| 건물 스킨 | 외형 변경 | $0.99~$2.99 |

### 3.4 꾸미기/테마 BM

#### 무료 구조
- 기본 건물 외형 무료
- 업적 달성으로 장식 해금
- 이벤트에서 한정 테마 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 식민지 테마 세트 | $6.99~$14.99 |
| 배경 변경 | 행성/환경 배경 | $2.99~$4.99 |
| 이펙트 팩 | 날씨/시각 효과 | $1.99 |

### 3.5 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 자원 2배, 건설 단축, 위기 지원 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $2.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금 클리어 가능 (시간 차이)
- 난이도 조절로 다양한 플레이어 대응
- 경쟁보다 개인 도전 중심

### 과금 압박 최소화
- 하드코어 유저는 도전 자체가 목적
- 시간 여유 있는 플레이어는 무과금으로 충분
- 위기/실패도 게임의 일부로 설계
