## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 시스템 밸런스 |
| 7~10주 | 난이도 프리셋 튜닝 |
| 11~12주 | 첫 챌린지 시즌 준비 |

**핵심 과제:**
- 하드코어 시스템 안정화
- 난이도 다양성 확보
- 장기 플레이 유도

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 난이도/룰 프리셋 이벤트 |
| 5개월 | 커스텀 룰 이벤트 |
| 6개월 | 장기 시나리오 업데이트 1차 |
| 7개월 | 하드코어 챌린지 시즌 |
| 8개월 | 장기 시나리오 업데이트 2차 |

**핵심 과제:**
- 커스텀 룰 시스템 검증
- 시나리오 콘텐츠 확장
- 챌린지 시스템 정착

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 랭킹/스테이지 시스템 |
| 10개월 | PC 포팅 준비 |
| 11개월 | 대규모 시나리오 업데이트 |
| 12개월 | 1주년 챌린지 챔피언십 |

**핵심 과제:**
- 랭킹 시스템 구축
- PC 포팅 기반 마련
- 하드코어 커뮤니티
