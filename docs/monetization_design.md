# game_0023 수익화 전략 - 콜로니 프론티어

## 게임 개요
- **장르**: 우주 식민지 경영 + 4X + JRPG
- **타깃**: 미드코어~하드코어, 경영/전략 선호
- **수익화 우선순위**: IAP > 광고

---

## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `resource_boost` | 자원 수확 | 자원 2배 | 5회 |
| `build_speed` | 건물 건설 중 | 시간 -30% | 5회 |
| `explore_bonus` | 행성 탐사 완료 | 보너스 보상 | 3회 |
| `research_speed` | 연구 진행 중 | 시간 -1시간 | 3회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |

### 1.2 전면 광고

- 경영 몰입 방해 최소화
- 쿨다운 5분
- 구독자 제외
