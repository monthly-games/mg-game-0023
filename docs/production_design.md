## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **콜로니 건설** | 미개척 행성에 콜로니 인프라 구축 | ★★★★★ |
| **생산 체인** | 복잡한 자원 생산 라인 최적화 | ★★★★★ |
| **생존 관리** | 산소, 식량, 에너지 등 생존 지표 관리 | ★★★★☆ |
| **행성 탐사** | 새로운 행성 발견 및 개척 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **최적화 완료**: 생산 체인 병목 해결
- **생존 안정화**: 모든 생존 지표 안정권 진입
- **행성 완료**: 새 행성 개척 성공
- **연구 완료**: 새 기술 해금

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 자원 수급 밸런스
├── 생존 지표 관리
├── 재난 대응 (운석, 폭풍, 질병)
├── 생산 체인 최적화
└── 인구 관리 및 행복도
```

### 1.4 사회적 요소 (Social)
- **리더보드**: 행성별 개척 속도 랭킹
- **연합**: 여러 플레이어 연합 도전
- **무역**: 플레이어 간 자원 교환
- **시드 공유**: 좋은 행성 시드 공유

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (1~3분) - 자원 수확
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [생산 확인] → [자원 수확] → [건물 배치]          │
│       ↑                            ↓             │
│       └──── [생존 체크] ← [업그레이드] ←┘        │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **생산 확인** (10초): 생산 현황 및 병목 확인
2. **자원 수확** (5초): 축적된 자원 수집
3. **건물 배치** (30초): 새 건물 건설
4. **업그레이드** (20초): 기존 건물 업그레이드
5. **생존 체크** (10초): 생존 지표 확인

### 2.2 세션 루프 (10~30분) - 전략 수립
```
[접속] → [방치 보상] → [생존 체크] → [생산 최적화] → [연구 진행]
                                              ↓
                              [재난 대응] ← [건물 건설]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [자원 생산] → [기술 연구] → [새 건물 해금]           │
│         ↓                              ↓             │
│  [행성 안정화]                   [프로브 발사]        │
│         ↓                              ↓             │
│  [다음 행성 개척] ←───────────────────┘              │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[방치 보상 화면]
    ↓
[콜로니 뷰] ─────→ [건물 메뉴]
    │              [연구 트리]
    │              [행성 맵]
    │              [설정]
    ↓
[건설] → [건물 선택] → [배치] → [생산 시작]
    │
    └── [탐사] → [프로브 발사] → [행성 발견] → [개척]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 35% | 생산 부스트, 재난 보험 |
| **프리미엄 패스** | 30% | 시즌 패스, 탐험가 패스 |
| **나노바이트** | 25% | 인앱 구매 화폐 |
| **편의 기능** | 10% | 시간 단축, 프리미엄 건물 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 기본 행성 개척 가능
├── 전체 연구 트리 접근
├── 시간 투자로 완전 진행
└── 광고 시청으로 부스트

과금 유저 혜택:
├── 나노바이트 패키지 ($0.99~$99.99)
├── 프리미엄 패스 ($9.99~$14.99)
├── 건축가 팩 ($6.99)
├── 탐험가 팩 ($9.99)
└── 광고 제거 ($4.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 나노바이트 (100) | $0.99 | 시간 단축 5회 |
| 중량 나노바이트 (600) | $4.99 | 25% 보너스 |
| 대량 나노바이트 (1500) | $9.99 | 50% 보너스 |
| 프리미엄 패스 | $9.99/시즌 | 전용 건물 + 보상 |
| 건축가 팩 | $6.99 | 프리미엄 건물 + 자원 |
| 탐험가 팩 | $9.99 | 프로브 + 희귀 자원 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 로그인 자원 보너스
- **생존 압박**: 정기적 접속 필요
- **주간 도전**: 특별 도전 과제
- **시즌**: 3개월 시즌 보상

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
복잡도
  ▲
  │                              ╭── 가스 행성 위성
  │                         ╭────╯ 화산 행성
  │                    ╭────╯ 빙하 행성
  │               ╭────╯ 사막 행성
  │          ╭────╯ 시작 행성
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 진행도
```

**난이도 공식:**
```
생존_난이도 = 행성_배율 × (1 + 재난_확률)
생산_복잡도 = 필요_자원_종류 × 체인_깊이
```

### 4.2 콘텐츠 진행
| 행성 유형 | 난이도 | 특징 | 해금 조건 |
|----------|-------|------|----------|
| 시작 행성 | 쉬움 | 튜토리얼, 기본 자원 | 시작 |
| 사막 행성 | 보통 | 물 부족, 태양 에너지 | 첫 행성 완료 |
| 빙하 행성 | 어려움 | 극저온, 희귀 광물 | 연구 20 완료 |
| 화산 행성 | 매우 어려움 | 지열 에너지, 귀금속 | 연구 50 완료 |
| 가스 행성 위성 | 극한 | 반물질, 이그조틱 | 연구 100 완료 |

### 4.3 밸런싱 공식
```dart

double calculateProduction(Building building, Colony colony) {
  double base = building.baseProduction;
  double techBonus = 1 + colony.techLevel * 0.02;
  double workerBonus = building.workers / building.maxWorkers;
  double powerBonus = colony.powerSupply >= 1.0 ? 1.0 : colony.powerSupply;
  return base * techBonus * workerBonus * powerBonus;
}

// 소비량 계산
double calculateConsumption(Colony colony, String resource) {
  double base = 0;
  for (var building in colony.buildings) {
    base += building.consumption[resource] ?? 0;
  }
  return base * colony.population / 100;
}

  double oxygen = min(1.0, colony.oxygenSupply / colony.oxygenDemand);

  double energy = min(1.0, colony.energySupply / colony.energyDemand);
  double water = min(1.0, colony.waterSupply / colony.waterDemand);
```

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 핵심 자원 (철, 물, 식량, 에너지)
├── 생존 지표 (산소, 온도)
├── 인구
└── 경고 알림

레벨 2 (상호작용 시):
├── 건물 상세
├── 생산 체인 뷰
├── 연구 트리
└── 행성 맵

레벨 3 (팝업):
├── 재난 경고
├── 연구 완료
├── 행성 발견
└── 설정 메뉴
```

### 5.2 네비게이션 흐름
```
[콜로니 뷰]
    ├── [건설] → [카테고리] → [건물 선택] → [배치]
    ├── [연구] → [연구 트리] → [연구 시작]
    ├── [탐사] → [프로브 발사] → [행성 발견]
    ├── [무역] → [자원 교환]
    ├── [상점] → [나노바이트/팩]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 건설 완료 | 건물 등장 | 건설음 | - |
| 생산 수확 | 자원 이펙트 | 수집음 | - |
| 연구 완료 | 해금 연출 | 발견음 | 진동 (중) |
| 재난 발생 | 경고 연출 | 경보음 | 진동 (강) |
| 행성 발견 | 탐사 연출 | 팡파레 | 진동 (강) |

### 5.4 접근성
- **생산 체인 시각화**: 자원 흐름 다이어그램
- **자동 알림**: 위기 상황 푸시
- **폰트 크기**: 3단계 조절
- **색맹 모드**: 자원 패턴 추가
- **자동 최적화**: 자동 건물 배치 제안
